# Advent

Solutions to [Advent of Code 2021](https://adventofcode.com/2021) in Elixir

## Usage

### Interactive

```shell
$ iex -S mix

iex(1)> import Advent.Day01
Advent.Day01
iex(2)> part1()
1266
iex(3)> part2()
1217
```

### Run Tests

Run tests on the example data sets given in the puzzles.

```shell
$ mix test
```
